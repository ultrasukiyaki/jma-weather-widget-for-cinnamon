# v3.0.0-alpha.2

## 地域設定UI刷新

- 都道府県と市区町村の連動セレクトを追加
- 気象庁の地域一覧をネットワーク取得し、ローカルへキャッシュ
- 市区町村から気象庁の発表官署コードと予報エリアを自動解決
- 気象庁予報JSONから気温地点名を自動判定
- Open-Meteo geocodingから緯度・経度を自動補完
- 緯度・経度の手動上書きに対応
- v2系の既存地域設定を可能な範囲で逆引きして移行
- Cinnamonの外部設定アプリ機構を使用したGTK3設定画面

## アーキテクチャ

- `LocationService`を追加し、UI設定からProvider設定への変換を分離
- 地域カタログ処理を`tools/location_catalog.py`へ分離
- 地域解析のスモークテストを追加

## 注意

ネットワークから地域一覧を取得できない場合、内蔵の最小フォールバックデータでは
函館市と東京都府中市のみ選択できます。取得成功後はキャッシュが利用されます。
