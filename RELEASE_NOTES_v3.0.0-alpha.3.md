# v3.0.0-alpha.3

v3の表示刷新段階です。気象庁・Open-Meteoの天気コードを`IconService`で統一し、パネル・現在天気・時間別予報・週間予報へ同梱SVGを表示します。

## 追加

- 14種類のフルカラーSVG天気アイコン
- JMA/Open-Meteoコード対応`IconService`
- パネルのSVGアイコン表示
- 現在天気の大型SVG表示
- 時間別・週間予報の行別SVG表示
- ポップアップアイコンサイズ設定
- SVG欠損時のテーマアイコンフォールバック

## 確認ポイント

- パネル表示3モードでSVGとラベルが正しく切り替わること
- 晴天時に昼夜アイコンが切り替わること
- ポップアップの時間別・週間予報にSVGが表示されること
- 設定画面のアイコンサイズ変更が再起動なしで反映されること
- 従来の地域変更・通知・外部リンクが壊れていないこと
