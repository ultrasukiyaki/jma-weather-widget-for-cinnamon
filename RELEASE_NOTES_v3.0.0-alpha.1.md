# v3.0.0-alpha.1 Release Notes

v3系の最初の検証版です。

## 主な変更

- JMAとOpen-MeteoのProvider化
- WeatherServiceによる取得統合
- WeatherSnapshotによる共通データ管理
- HTTP通信の分離
- API片方が失敗した場合の前回成功データ維持
- 設定画面互換修正を本体へ統合

## 表示・設定

表示内容と設定項目はv2.1.0と同等です。都道府県・市区町村セレクトとSVG表示は後続alphaで追加します。

## 注意

Cinnamon実機での確認が必要な開発版です。安定版利用者はv2.1.0を維持してください。
