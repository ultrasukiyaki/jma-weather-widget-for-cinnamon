# JMA Weather Japan 1.0

Cinnamon 6.6 / GJS 1.80向け、日本の天気予報アプレットです。

## 主な機能

- 気象庁の公開JSONを利用
- 今日の天気・最低/最高気温・降水確率・風
- 7日間の週間予報
- パネル表示を3種類から選択
- 雨通知
- 予想最高気温に基づく簡易暑さ警告
- 任意の府県予報区コード・予報エリアを設定可能
- tenki.jpなど任意の詳細ページをブラウザで開く

## インストール

```bash
unzip jma-weather-widget-for-cinnamon.zip
cd jma-weather-widget-for-cinnamon
./install.sh
```

その後、Cinnamonの「アプレット」から `JMA Weather Japan` を追加します。

既に0.1を使っている場合も、同じ手順で上書きできます。追加済みアプレットは
一度パネルから外して再追加するか、X11なら `Alt+F2` → `r` → Enterで再読込してください。

## 府中市の初期設定

- 府県予報区コード: `130000`
- 予報エリア名: `東京地方`
- 気温観測地点名: `東京`
- 表示地域名: `府中市`

気象庁の短期予報は市区町村そのものではなく「東京地方」単位です。
詳細な1時間予報は、設定したtenki.jp URLを開いて確認できます。

## 暑さ警告について

現在の暑さ警告は、公式の熱中症警戒アラートではありません。
気象庁の予想最高気温が設定値以上になった場合の簡易警告です。

## ログ確認

```text
Alt+F2 → lg → Enter
```

または:

```bash
journalctl --user -f | grep -i jma-weather
```
